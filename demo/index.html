Default gradient

No options – completely transparent

<poly-gradient width="320px" height="240px"></poly-gradient>

 

Colorful

Using the stops attribute

<poly-gradient angle="-90" stops='{"0":"#f00","1":"#0f0"}' width="320px" height="240px"></poly-gradient>

 

Colorful

Using <poly-gradient-stop> elements

<poly-gradient angle="-75" width="320px" height="240px">
    <poly-gradient-stop offset="0" color="#f00"></poly-gradient-stop>
    <poly-gradient-stop offset="0.15" color="#ff0"></poly-gradient-stop>
    <poly-gradient-stop offset="0.3" color="#0f0"></poly-gradient-stop>
    <poly-gradient-stop offset="0.5" color="#0ff"></poly-gradient-stop>
    <poly-gradient-stop offset="0.65" color="#00f"></poly-gradient-stop>
    <poly-gradient-stop offset="0.8" color="#f0f"></poly-gradient-stop>
    <poly-gradient-stop offset="1" color="#f00"></poly-gradient-stop>
</poly-gradient>

 

With content

<poly-gradient width="320px" height="240px" style="display:flex;align-items:center;justify-content:center;text-align:center">
    <poly-gradient-stop offset="0" color="#0B486B"></poly-gradient-stop>
    <poly-gradient-stop offset="1" color="#F56217"></poly-gradient-stop>

    <p style="color:#fff">Gradient elements can also have any type of content inside.</p>
</poly-gradient>

Gradient elements can also have any type of content inside.

 

From a preset

You can use a preset name instead of manually specifying stops

By default, presets are loaded from this JSON. A list of presets (names and preview) is available here: uigradients.com.

<poly-gradient preset="Sunset" width="320px" height="240px"></poly-gradient>

 

Nested

You can even nest gradients into each other

<poly-gradient angle="45" preset="Bloody Mary" width="320px" height="240px" style="display:flex;align-items:center;justify-content:center;text-align:center">
    <poly-gradient angle="-45" preset="Horizon" width="240px" height="120px"></poly-gradient>
</poly-gradient>
